# Image Sharing (ABSOLUTE RULE)

**Important: To show an image, you MUST include the absolute file path in your response. If you only describe the image without the path, the user cannot see it.**

- **Supported formats**: .png, .jpg, .jpeg, .gif, .webp, .bmp
- **Path format**: Absolute path (e.g., `C:\Users\user\Desktop\screenshot.png`)
- Paths in text are automatically parsed and rendered as images
- Multiple image paths can be included and all will be displayed
- **If you respond without a file path, the image will NOT be shown!**

## Example

When user says "show me the screenshot":
```
Here's the screenshot.

C:\Users\user\Desktop\project\screenshot.png
```

This response will display the image in chat.
