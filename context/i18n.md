# i18n (Internationalization)

Rico uses svelte-i18n based multi-language system. Claude naturally responds in the user's language, so no special language enforcement is needed.

## Supported Languages

| Code | Language |
|------|----------|
| ko | Korean |
| en | English |

## File Structure

```
web/src/
├── lib/i18n.ts          # i18n initialization and settings
└── locales/
    ├── ko.json          # Korean translations (~120 keys)
    └── en.json          # English translations
```

## How It Works

1. **Language Detection Priority**
   - localStorage saved value (`rico_locale`)
   - Browser language (`navigator.language`)
   - Default: `en`

2. **Usage in Components**
   ```svelte
   <script>
     import { _ } from 'svelte-i18n';
   </script>

   <button>{$_('chat.send')}</button>
   ```

3. **Language Switching**
   - Toggle button in SessionListScreen settings menu
   - Call `setLocale('ko')` or `setLocale('en')`
   - Auto-saved to localStorage

## Components with i18n Applied

- ChatScreen.svelte
- LoginScreen.svelte
- SessionListScreen.svelte
- SoulScreen.svelte
- SoulListScreen.svelte
- SoulEditScreen.svelte

## Adding a New Language

1. Create `{language_code}.json` in `web/src/locales/` folder
2. Copy structure from `ko.json` and translate
3. Add import and `addMessages()` in `i18n.ts`
4. Update language toggle UI
